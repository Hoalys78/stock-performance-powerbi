# 📈 Portfolio Financeiro \| Power BI

Sistema completo de análise quantitativa de investimentos desenvolvido
em **Power BI**, utilizando **DAX**, **Power Query**, modelagem
dimensional e conceitos de Business Intelligence.

## Objetivos

-   Demonstrar conhecimentos em Power BI, DAX e BI.
-   Aplicar indicadores financeiros e análise quantitativa.
-   Construir um dashboard executivo para acompanhamento de carteiras.

## Tecnologias

-   Power BI
-   DAX
-   Power Query (M)
-   Python
-   Excel
-   Git / GitHub

## Estrutura

``` text
Portfolio Financeiro
```

## Indicadores

-   Retorno Diário, Mensal, Anual e Acumulado
-   CAGR
-   Volatilidade
-   Drawdown
-   Sharpe Ratio
-   Alpha e Beta
-   Tracking Error
-   Information Ratio
-   Comparação com IBOV e CDI

-   Performance
